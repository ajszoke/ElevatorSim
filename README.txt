This is a command-line project designed to simulate an elevator control system in Java. It is a timing simulation that has the ability to queue called floors, and can be interrupted by a "fire key" and put into manual control. All instructions are provided inline within the console.

1. Download the most recent Java JDK from http://www.oracle.com/technetwork/java/javase/downloads/index.html

2. Run the following lines from the command line within the project directory:

javac *.java
java Driver
